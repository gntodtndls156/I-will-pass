CPU


CPU (Central Processing Unit)는 컴퓨터의 중앙처리장치로, 중앙에서 모든 데이터를 처리하는 장치이다.

컴퓨터의 가장 중요한 부품 중 하나로, 사람의 뇌와 같은 역할을 한다고 할 수 있다.

사용자로부터 명령어를 입력 받아, 해석하고 연산한 후 결과를 출력한다.

제조사에 따라 종류가 다르지만, 숫자가 높을수록 사양이 좋은 것은 동일하다.

RAM


RAM (Random Access Memory)은 컴퓨터의 주기억장치로, 보조기억장치(HDD, SSD)의 데이터를 일시적으로 불러와 중앙처리장치(CPU)에서 작업을 할 수 있도록 도와준다.

기억된 정보를 읽어내거나 다른 정보를 기억시킬 수 있는 메모리이며, 4GB, 8GB, 16GB 등의 종류가 있다.

주로 단순 문서작업을 한다면 8GB를, 이외의 더 무거운 작업을 한다면 16GB를 사용하는 것이 좋다.

위로


HDD


HDD (Hard Disk Drive)는 컴퓨터의 보조기억장치로, 컴퓨터의 정보, 문서, 자료 등을 저장하고 읽을 수 있다.

내장 하드 디스크는 컴퓨터의 본체에 부착된 것을, 외장 하드 디스크는 본체에 연결하여 사용할 수 있는 것을 이야기한다.

디스크를 빠르게 회전시켜 저장하는 방식으로, 소음이 발생하고, 충격에 약하며, 전력소모가 높아 최근에는 디스크가 아닌 메모리를 사용하는 SSD를 많이 사용한다.

그러나 많은 양의 자료를 보관하는 사용자에게는 HDD가 유용하다.

위로


SSD


SSD (Solid State Drive)는 컴퓨터의 보조기억장치 중 하나로, 반도체를 기반으로 정보를 저장하는 장치이다.

하드디스크(HDD)보다 속도가 빠르고, 소비전력이 낮아 최근 대체품으로 많이 사용되고 있으며, 소음과 발열이 적고, 충격에 강하다는 장점이 있다.

따라서 SSD의 사용은 배터리 유지시간을 늘릴 수 있다. 그러나 용량 대비 가격이 하드디스크에 비해 비싸다.

위로


그래픽카드


그래픽카드(Graphics Card)는 컴퓨터의 디지털 데이터를 영상신호로 바꾸어 모니터로 전송하는 장치이다.

간단한 작업이나 사무용으로 이용하는 경우에는 컴퓨터에 이미 포함되어 있는 온보드 혹은 내장 그래픽을 사용하고,

게임, 그래픽 작업을 주로 이용하는 사용자는 고성능의 그래픽카드를 따로 사용한다.

그러나 고성능일수록 소비전력과 발열이 높고 가격이 비싸다.

또한, 데스크탑용 그래픽카드와 노트북용 그래픽카드는 서로 호환되는 것이 다르므로 확인 후 구매하여야 한다.

보통 노트북 사용자는 본체의 크기 및 발열 문제로 내장형 그래픽을 많이 사용한다.
